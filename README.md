# 🎮 Projeto Player

Este projeto simula um sistema básico de um jogador de RPG utilizando PHP, com funcionalidades de ataque, defesa, inventário e magia.

---

## 📁 Estrutura do Projeto

- `index.php`: Arquivo principal que executa o sistema.
- `Player.php`: Classe principal que representa o jogador.
- `Ataque.php`: Gerencia ações ofensivas.
- `Defesa.php`: Gerencia ações defensivas.
- `Inventario.php`: Lida com os itens do jogador.
- `Item.php`: Define os objetos que podem ser adicionados ao inventário.
- `Magia.php`: Lida com magias e habilidades especiais.
